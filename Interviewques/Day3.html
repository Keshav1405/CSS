CSS interview theoretical questions

Q1. Difference between flex row and column.

Both of them comes under flex-direction property when the HTML element is given display:flex . Flex row allows developer to align items along the row in which the main axis is horizontal & cross axis is vertical. Flex column allows developer to align items along the column in which the main axis is vertical & cross axis is horizontal.

Q2. Explain inline , internal and external stylesheet.

Inline stylesheet means the styling is given in the opening tag for each & every element. Internal stylesheet means the styling is given inside the head tag of HTML file using style tag, an element is targeted using simple selector. External stylesheet means the styling is given in a separate file with .css extension which is linked to the html file using the link tag inside the head tag of html file. Inline styling will always takes preference over the others.

Q3. Justify-content allows you to do what?

Justify-content is a property of both flexbox and grid in css. It tells browser how to distribute the content items along the main axis. It has many values like flex-start, flex-end, center, space-between, space-around, space-evenly. Its default is Justify-content: flex-start

Q4. Difference between absolute and relative positioning.

"Absolute positioning" means the element is removed from the flow & is relatively positioned to its first non-static ancestor or its parent element whereas "relative positioning" means the element is positioned relative to its normal position.

Q5. What is grid-template-columns used for?

Grid-template-columns is used to specify the number of columns and their individual width in a grid layout. For example:- grid-template-columns: 10px 20px auto 40px

This grid will have 4 columns with 10px, 20px, auto, 40px width respectively.

Q6. What is the z-index in CSS?

The z-index (3rd dimension) property specifies the stack order of an element in CSS. It defines which layer will be above in case of overlapping element. For example:- z-index: 1

Q7. What is the difference between padding and margin?

Padding is the space between the element's content and border while margin is space around the border of an element. Margin can have negative values in which case the elements will overlap whereas padding can't be given a negative value. Margin is an invisible boundary around the border of an element while padding takes the background color of the content.

Q8. What is box sizing?

Box-sizing is a property in CSS which determines whether padding & border should be included in element's width & height. The default value of box-sizing is "content-box" where the content width & height is the element's width & height, the other value of box-sizing is "border-box" where element's width & height include the content width & height + padding + border.

        Box-sizing: content-box;
        Box-sizing: border-box;

Q9. What is animation delay?

Animation delay is a property which specifies the delay for the start of an animation. If the animation is repeating, the delay will occur only once. Its value is given in seconds or miliseconds. For example:- animation-delay: 2s

Q10. Which property you will use to merge cells vertically in grid?

We can use "grid-row" property to merge cells vertically, it should be given to the required cell. For example:- grid-row: 1 / 3. Here the cell will span from first row to third row.

Q11. Which property you will use to merge cells horizontally in grid?

We can use "grid-column" property to merge cells horizontally, it should be given to the required cell. For example:- grid-column: 1 / 3. Here the cell will span from first column to third column.

Q12. Explain CSS box model.

CSS box model looks at all the HTML elements as rectangular boxes. It is used for design & layout of an element. It consists of margin, border, padding and the actual content. CSS box model can also be defined as a set of rules that determine how a webpage is rendered on the internet.

Q13. what is the difference between display none and visibility hidden?

Display none will hide the element and remove it from the flow of document that means the next element will take its space whereas visibility hidden will hide the element but it will retain its space in the document & the next element will be positioned after it.
